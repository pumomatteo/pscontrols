✅ Approccio Consigliato: PSButton React

CONSIGLIATO

Usa il componente React PSButton per tutti i progetti moderni. Fornisce funzionalità avanzate, accessibilità e stili ottimizzati.

📦 Come Usarlo:

// 1. Include PSControls React build
<script src="./dist/react/pscontrols-react.min.js"></script>
<link rel="stylesheet" href="./dist/react/pscontrols-react.min.css">

// 2. JavaScript/JSX (con Babel o TypeScript)
const { PSButton, ButtonModeEnum, PSControlsProvider } = window.PSReact;

<PSControlsProvider>
  <PSButton 
    text="My Button"
    mode={ButtonModeEnum.Primary}
    onClick={() => alert('Clicked!')}
    tooltip="Tooltip text"
    confirmationMessage="Are you sure?"
  />
</PSControlsProvider>

🎯 Vantaggi:

  • ✅ Tooltip automatico
  • ✅ Conferme integrate
  • ✅ Stati loading
  • ✅ Accessibilità completa
  • ✅ Temi e modalità colore
  • ✅ Badge e icone

⚡ Alternativa: Button HTML + CSS PSControls

ALTERNATIVA

Usa button HTML normale con classi CSS PSControls. Ideale per progetti senza React o per controllo granulare.

📦 Come Usarlo:

// 1. Include solo CSS PSControls
<link rel="stylesheet" href="./dist/vrcontrols.all.min.css">

// 2. HTML normale con classi CSS
<button class="vrButtonDefaultMode" onclick="myFunction()">
  Default Button
</button>

<button class="vrButtonPrimaryMode" onclick="alert('Primary!')">
  Primary Button
</button>

<button class="vrButtonDangerMode" title="Tooltip manuale">
  Danger Button
</button>

🎯 Classi CSS Disponibili:

  • .vrButtonDefaultMode - Bottone default
  • .vrButtonPrimaryMode - Bottone primary (blu)
  • .vrButtonSuccessMode - Bottone success (verde)
  • .vrButtonDangerMode - Bottone danger (rosso)
  • .vrButtonWarningMode - Bottone warning (arancione)
  • .vrButtonDeleteMode - Bottone delete
  • .vrButtonExcelMode - Bottone Excel
  • .vrButtonPrintMode - Bottone print

🔄 Migrazione: createReactButton()

MIGRAZIONE

Helper per migrazione rapida da vr.createButton(). Mantiene la stessa sintassi del vecchio sistema.

📦 Come Usarlo:

// 1. Include PSControls legacy build
<script src="./dist/vrcontrols.all.min.js"></script>

// 2. JavaScript normale (no React diretto)
// Sostituisci il vecchio vr.createButton():
createReactButton({
  text: "My Button",
  mode: ButtonModeEnum.Primary,
  onClick: () => alert('Clicked!'),
  tooltip: "My tooltip",
  confirmationMessage: "Are you sure?"
}, document.getElementById('my-container'));

🎯 Quando Usarlo:

  • ✅ Migrazione veloce da codice esistente
  • ✅ Nessuna modifica alla logica esistente
  • ✅ Compatibilità con vr.createButton()
  • ❌ Meno flessibile di PSButton puro

🔧 Complete Props Interface

interface PSButtonProps {
  // Text & i18n
  text?: string;
  textKey?: string;           // i18n translation key
  textParams?: object;        // i18n parameters
  
  // Appearance
  mode?: ButtonModeEnum;
  colorSettings?: ColorSettings;
  width?: string | number;
  height?: string | number;
  className?: string;
  style?: React.CSSProperties;
  
  // Icon & Image
  icon?: IconClass;           // CSS class for icon
  onlyIcon?: boolean;         // Hide text, show only icon
  imageUrl?: string;          // Image instead of icon
  
  // Badge
  badgeSettings?: {
    text?: string;
    visible?: boolean;
    color?: string;
    backgroundColor?: string;
  };
  
  // State
  loading?: boolean;
  enabled?: boolean;
  visible?: boolean;
  
  // Interaction
  value?: string;
  tooltip?: string;
  tooltipKey?: string;        // i18n tooltip key
  confirmationMessage?: string;
  confirmationMessageKey?: string;
  
  // Events
  onClick?: (event: React.MouseEvent) => void;
  onRightClick?: (event: React.MouseEvent) => void;
  onMiddleClick?: (event: React.MouseEvent) => void;
  onHover?: (event: React.MouseEvent) => void;
  onBlur?: (event: React.FocusEvent) => void;
  onMouseDown?: (event: React.MouseEvent) => void;
  onMouseUp?: (event: React.MouseEvent) => void;
  
  // Accessibility  
  'aria-label'?: string;
  'aria-describedby'?: string;
  'aria-expanded'?: boolean;
  'aria-pressed'?: boolean;
  role?: string;
  tabIndex?: number;
  testId?: string;            // For testing
}
